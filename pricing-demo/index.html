AliExpress Markup Formula

Every product fetched from AliExpress automatically has a 20% markupapplied before being displayed on LuxeShop. Here's exactly how it works.

The Pricing Formula

AliExpress Price

P

Raw price from AliExpress API

Markup Factor

× 1.20

20% service markup

LuxeShop Price

= P × 1.20

Price shown to customers

// aliexpress.ts — applyMarkup()
const MARKUP_PERCENT = 20;
function applyMarkup(aliPrice: number) {
const ourPrice = parseFloat((aliPrice * (1 + MARKUP_PERCENT / 100)).toFixed(2));
const profitPerSale = parseFloat((ourPrice - aliPrice).toFixed(2));
return { ourPrice, profitPerSale };
}

Integration Architecture

1

API Request

Frontend calls /api/aliexpress/search or /hot-products

2

Fetch from AliExpress

Next.js server calls RapidAPI (AliExpress True API) with your key

3

Apply Markup

applyMarkup() multiplies every price by 1.20 (20% markup)

4

Return to UI

Normalised LuxeProduct objects with ourPrice and profitPerSale

Ready for Production

Add your RAPIDAPI_KEY to .env.local to switch from demo mode to live AliExpress data. The markup formula, caching, and error handling are all production-ready.
